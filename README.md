ASTU Smart Complaint System
📌 Project Overview

This ASTU Smart Complaint System is a frontend-based web application developed using HTML, CSS, JavaScript, and Bootstrap.
The system allows:
- Students to submit complaints
- Staff to manage and update complaint status
- Admin to monitor system activity
- Users to receive notifications when complaint status changes

=> This project simulates a real complaint management system using LocalStorage (frontend-only), without a backend server.

🛠 Technologies Used
- HTML
- CSS
- Bootstrap 
- JavaScript
- LocalStorage (for data simulation)
